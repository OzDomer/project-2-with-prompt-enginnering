# Cryptonite

A cryptocurrency dashboard that displays real-time data on the 100 most popular crypto coins, live price charts for user-selected coins, and AI-powered buy/sell recommendations.

## Features

- Browse 100 popular cryptocurrencies with search and filtering
- Bookmark up to 5 coins for tracking
- Real-time price charts for bookmarked coins
- AI recommendations (buy or don't buy) powered by OpenAI

## Tech Stack

React, TypeScript, Vite, Redux Toolkit, react-router-dom, axios, recharts, plain CSS

## Getting Started

Requires [Node.js](https://nodejs.org/) installed.

```
npm install
npm run dev
```
